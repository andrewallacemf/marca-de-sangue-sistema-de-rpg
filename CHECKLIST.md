# ✅ Checklist de toda sessão de trabalho

*Versão de 1 página para imprimir ou deixar aberta. Detalhes: [manual 04](manuais/04-fluxo-de-trabalho.md).*

> ### 🟢 PUXE no começo. SUBA no fim.

```
ANTES
 1. [ ] PULL       GitHub Desktop → Fetch origin → Pull origin
 2. [ ] AVISO      no grupo: "vou mexer em X"

DURANTE
 3. [ ] EDITAR     você ou a IA (IA? peça pra ela ler AGENTS.md primeiro)

DEPOIS
 4. [ ] CONFERIR   aba Changes: leia o que mudou (você é o revisor da IA)
 5. [ ] REGISTRO   usou IA? confira/crie o arquivo em registro-ia/ANO/
 6. [ ] PENDÊNCIAS ficou algo "A DEFINIR"? confira se entrou no PENDENCIAS.md
 7. [ ] ÍNDICE     mexeu em conceito/regra? confira se o INDICE.md foi atualizado
 8. [ ] COMMIT     frase curta em português do que você fez → Commit to main
 9. [ ] PUSH       Push origin
10. [ ] AVISO      no grupo: "terminei X, já subi"
```

**Deu erro, conflito ou algo sumiu?** Não tente adivinhar: [conflitos](manuais/06-conflitos.md) ·
[problemas comuns](manuais/07-problemas-comuns.md). Em último caso, chame o guardião do Git no grupo.

**Nunca faça** (nem deixe uma IA fazer sem todos saberem): forçar push (`--force`),
apagar a pasta do projeto para "baixar de novo" sem antes subir o que tinha, ou editar
sem ter dado pull.
