# ⚙️ Sistema-base

As **regras universais** do RPG — as que valem para qualquer cenário. Tudo o que é
específico de um mundo (Shogunato, Junkpunk, Ratos…) mora em `../cenarios/`, não aqui.

## Como está organizado

| Pasta | Conteúdo |
|-------|----------|
| `conceitos/` | Fundamentos: o que é o jogo, como se lê uma regra, termos centrais. |
| `criacao-de-personagem/` | Passo a passo de criação, atributos, escolhas iniciais. |
| `testes-e-combate/` | Resolução de ações, rolagens, combate, condições e estados. |
| `progressao/` | Evolução de personagem, recompensas, marcos. |
| `itens/` | Itens genéricos disponíveis em qualquer cenário. |
| `criaturas/` | Criaturas genéricas (bestiário base). |

## Regra de ouro do sistema-base

Se uma regra só faz sentido em **um** mundo, ela **não** é do sistema-base — é de cenário.
O base é o esqueleto comum sobre o qual os cenários se apoiam.

## Como carregar a v1

Traga o conteúdo da sua primeira versão para as pastas acima, quebrando em arquivos
pequenos (um assunto por arquivo) e seguindo o [CONVENCOES.md](../CONVENCOES.md). Use os
`_template-*.md` como molde para itens e criaturas.

> ⚠️ A DEFINIR: substituir estes templates pelo conteúdo real da v1 do sistema.
