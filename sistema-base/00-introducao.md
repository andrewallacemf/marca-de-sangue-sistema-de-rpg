# Introdução

Este documento reúne as regras básicas e gerais sobre o sistema Marca de Sangue. Aqui você vai conhecer os mecanismos de funcionamento do jogo que se aplicam à maioria dos cenários que você imaginar, seja uma jornada medieval clássica, ou uma epopeia espacial a bordo de uma nave futurística.

Mas como todo tipo de história merece ser vivida de uma forma particular, dentro do sistema Marca de Sangue, cada cenário tem mecânicas únicas que interagem com as regras básicas, tornando cada jogo uma experiência singular. Sinta-se livre para criar seus próprios insumos para o jogo, mas, para facilitar, aqui você também encontra uma série de cenários criados pelos idealizadores do jogo e fortemente sugeridos para uma experiência inicial.

Dentro de cada cenário da lista de cenários estão as regras, mecânicas específicas, informações de mundo e um acervo de inimigos que podem ser inseridos em aventuras contadas nesses mundos.

O sistema é construído para ser jogado em uma malha hexagonal. Contudo, o jogo é facilmente adaptável para que você use qualquer papel quadriculado.

Ao fim do Manual base do jogador há uma seção dedicada ao **passo-a-passo de criação de personagens para jogadores.** É recomendado ler as demais seções antes de criar seu personagem.

## O que preciso para jogar

Para jogar RPG utilizando o Marca de Sangue é necessário:

> 💡 **Um grupo de amigos dispostos a jogar**
> *Recomenda-se de 3 a 6 pessoas.*

> 💡 **Dados de 10 faces, conhecidos como d10**
> *Recomenda-se pelo menos 2 dados de 10 faces (2d10) por pessoa.*

> 💡 **Dados de 4, 6, 8, 10, 12 e 20 faces**
> *Conhecidos como d4, d6, d8, d10, d12 e d20, respectivamente.*

> 💡 **Malha de combate físico**
> *1 malha de combate físico hexagonal para representar o posicionamento dos jogadores e NPCs em combate.*
> **Alternativas:** Pode-se utilizar a malha de combate físico quadriculada, padrão dos RPGs de mesa.

> 💡 **Fichas do sistema Marca de Sangue**
> *1 ficha por pessoa. Podem ser baixadas gratuitamente e impressas.*
> **Alternativas:** Para uma experiência mais completa, use as fichas do cenário específico.

## Como começar jogar

1. **Se familiarize com as regras do sistema.** Pelo menos 1 pessoa do grupo precisa conhecer as regras básicas do jogo.
2. **Adquira/separe os materiais indicados para jogar.** Imprima as fichas dos jogadores e a malha de combate físico, separe dados (principalmente d10). Caso não seja possível, papel em branco, lápis e um celular com um aplicativo de rolagem de dados é suficiente.
3. **Marque uma sessão de jogo** (online ou presencial) com seu grupo, recomenda-se separar no mínimo 4 horas para jogar. No primeiro encontro pode ser necessária uma hora extra para criar os personagens dos jogadores.
4. **Defina com seu grupo quem será o narrador,** responsável por preparar a aventura e representar o mundo e os personagens que não são dos jogadores.
    1. **A pessoa designada como Narrador** deve escolher um cenário pronto ou criar um próprio e preparar uma sessão (com contexto de jogo, objetivo do grupo e desafios possíveis que eles poderão enfrentar) com antecedência;
    2. **Os demais jogadores devem criar seus personagens** em uma ficha, seguindo as regras descritas no Manual base do jogador e compartilhá-la com o narrador.
5. **Joguem o jogo**, o narrador deve inserir os personagens em uma aventura e os jogadores devem interpretar seus personagens para juntos vencerem os desafios preparados pelo narrador.
