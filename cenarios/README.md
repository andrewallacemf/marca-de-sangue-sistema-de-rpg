# 🌍 Cenários

Cada cenário é um **livro** que se apoia no `../sistema-base/` e acrescenta o que é
exclusivo daquele mundo: mecânicas próprias, itens, criaturas, locais e lore.

## Princípio central

Um cenário **não reescreve** o sistema-base. Ele **referencia** a regra base e descreve
apenas a **diferença** — o que muda e o que adiciona. Ver [CONVENCOES.md](../CONVENCOES.md),
seção 4.

## Criar um cenário novo

1. **Copie a pasta inteira** `_template-cenario/`.
2. Renomeie a cópia seguindo as convenções (minúsculas, sem acento, com hífen).
3. Preencha a partir dos templates de dentro dela.
4. **Não escreva** dentro do `_template-cenario/` original — ele é só molde.
