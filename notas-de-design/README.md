# 🧪 Notas de design

Aqui mora o que **não é regra vigente**, mas explica ou alimenta o design do sistema:

- **[variantes-pendentes.md](variantes-pendentes.md)** — versões alternativas de regras
  encontradas durante a migração do Notion (as antigas colunas "Sugestões"), preservadas
  até o grupo decidir caso a caso.
- **[hipoteses-de-design.md](hipoteses-de-design.md)** — ideias e apostas de design em
  discussão.

Regra de uso: quando o grupo decidir uma pendência de regra, a decisão vai para o
arquivo de regra correspondente (em `sistema-base/` ou `cenarios/`), a variante
descartada ganha uma linha em "decisões tomadas" na nota, e a pendência sai do
[PENDENCIAS.md](../PENDENCIAS.md).
